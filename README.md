## nodejs-local-api

使用 Express 4.x + Nodejs + ejs + Bootstrap 开发的一套简易的利用本地json文件做数据库的API系统Demo。

## 开发环境配置

- Node.js: `6.4.0`
- Express：`4.14.1`
- ejs: `2.5.6`

## 项目运行

- npm install
- npm run dev
- api 管理:  http://localhost:3100/main
- api 访问:  http://localhost:3100/zhoou/(api-name)
